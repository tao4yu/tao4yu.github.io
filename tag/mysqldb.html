Python MySQLdb Startup

 2013-12-27 by Tao4Yu

因项目实际需要，须利用Python在MySQl数据库中建立中文的表名及中文字段名（很诡异的需求），出现的各种问题让人头疼，现将过程记录一下。 开发环境如下：

  • Win8 x64
  • python 2.7
  • virtualenv 1.10
  • mysql5.5

一、建立项目目录

通过如下命令建立项目目录env1，并激活该目录：

cd virtualEnv
virtualenv   env1
cd env1
cd scripts
activate.bat

二、下载并安装Python MySQLdb库

  • 下载对应64位的mysqldb库。 我下的是：MySQL-python-1.2.3.win-amd64-py2.7.exe。 将下载的安装文件放至env1目录下，在env1目录下(已激活)，通过如下命令安装安装。

    :::sh easy_install MySQL-python-1.2.3.win-amd64-py2.7.exe

  • 测试是否安装成功，运行命令python进入隔离环境env1。

    :::python import MySQLdb

如果没报错，说明安装成功。

三、建立数据库表（已经建立名为testdb的数据库，数据库编码为utf-8）

建立表名及字段名为中文的数据库表。

# ! /usr/bin/env python
# -*- coding: utf-8 -*-

# 在MySQL中建立中文表名及中文字段名
# MySQLdb 默认编码为 latin-1，改为utf-8
import MySQLdb as mdb

con = mdb.connect('localhost', 'root', 'root', 'testdb')
con.set_character_set('utf8')

with con:
    cur = con.cursor()
    # 设置utf8参数
    cur.execute('set names utf8')
    cur.execute('set character set utf8')
    cur.execute('set character_set_connection=utf8')

    cur.execute("create table 中文表名(Id int primary key auto_increment, 名称 text)")

执行该段 python代码，即可在数据库中建立中文的表名及字段名。
